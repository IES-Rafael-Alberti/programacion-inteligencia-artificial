## 1. Carga y comprensión del dataset (25%)

| Nivel | Descripción |
| --- | --- |
| Insuficiente | No identifica bien el dataset ni su estructura. |
| Básico | Comprende parte del dataset, pero con lagunas relevantes. |
| Adecuado | Describe correctamente origen, tipo y variables principales. |
| Excelente | Interpreta el dataset con claridad y contexto. |

## 2. Limpieza y transformación (30%)

| Nivel | Descripción |
| --- | --- |
| Insuficiente | No limpia ni transforma correctamente los datos. |
| Básico | Aplica limpiezas parciales o poco justificadas. |
| Adecuado | Realiza limpieza y transformación adecuadas. |
| Excelente | Aplica limpieza rigurosa y decisiones bien justificadas. |

## 3. EDA, visualización y análisis (25%)

| Nivel | Descripción |
| --- | --- |
| Insuficiente | No realiza un EDA útil ni visualizaciones comprensibles. |
| Básico | Presenta análisis parcial con interpretaciones limitadas. |
| Adecuado | Realiza EDA y visualización correctos y útiles. |
| Excelente | Extrae conclusiones claras y bien sustentadas. |

## 4. Reproducibilidad, joins y groupby, presentación (20%)

| Nivel | Descripción |
| --- | --- |
