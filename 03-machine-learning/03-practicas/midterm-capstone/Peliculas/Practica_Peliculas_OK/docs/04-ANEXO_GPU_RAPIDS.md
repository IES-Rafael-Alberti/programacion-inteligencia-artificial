# Anexo — Uso de GPU con cuDF y cuML (alternativa)

Este anexo es voluntario y no penaliza no realizarlo.

## Objetivo
Comparar el uso de CPU frente a GPU en tareas de análisis y modelado.

## Herramientas
- cuDF (sustituto de pandas)
- cuML (modelos ML acelerados por GPU)

## Tareas propuestas
- Repetir parte del EDA con cuDF
- Entrenar uno o dos modelos con cuML
- Comparar tiempos y métricas

## Conclusiones esperadas
- ¿Dónde mejora el uso de GPU?
- ¿Dónde no aporta ventaja?
- ¿Merece la pena según el tamaño del dataset?
