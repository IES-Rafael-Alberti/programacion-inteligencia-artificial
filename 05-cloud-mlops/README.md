# 05-cloud-mlops

## Propósito
Cubre conceptos de cloud computing, MLOps, feature stores, infraestructura cloud para IA, despliegue de modelos, RAG en cloud, y herramientas de orquestación.

## Materiales incluidos
- **01-teoria**: Documentación completa (14 guías en MD+HTML) sobre feature stores, cloud compute, MLOps, almacenamiento IA, frameworks ML, APIs LLM, orquestación, herramientas datos, vectores, serving, RAG, stacks IA
- **02-ejemplos**: Demos de RAG (sencillo, Ollama, PageIndex), pipelines de feature/training/inference con Hopsworks
- **03-practicas**: Tarea comparativa de herramientas cloud + rúbrica. Rescatado de 24-25: scriptAWS.sh, esqueletoAWS.py, tarea AWS (org+pdf)
- **04-evaluacion**: Cuestionarios GIFT (infraestructura datos, IA generativa RAG)
- **05-recursos**: predict_example.py, quickstart_fraud_model, indexed_docs.json
- **90-archivo**: herramientasCLoudPIA.zip (pendiente de revisar contenido)
- **99-profesor**: modelos-gpt-5.4.md
