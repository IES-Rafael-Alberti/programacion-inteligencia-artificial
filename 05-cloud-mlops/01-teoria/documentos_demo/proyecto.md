# Proyecto de ejemplo

## Descripción

Este proyecto consiste en desarrollar una aplicación de apoyo al alumnado para consultar información del curso, resolver dudas frecuentes y acceder a orientaciones sobre entregas, evaluación y organización del trabajo.

## Objetivos

- Centralizar información del proyecto en un único punto.
- Facilitar consultas rápidas mediante búsqueda y asistencia conversacional.
- Reducir dudas repetitivas sobre entregas, criterios y fases del trabajo.

## Usuarios

- Alumnado del módulo.
- Profesorado que quiera reutilizar materiales o responder preguntas frecuentes.

## Funcionalidades previstas

- Consulta de documentación del proyecto.
- Respuestas a preguntas frecuentes.
- Resumen de requisitos de entrega.
- Apoyo para localizar información relevante en varios documentos.

## Restricciones

- El tiempo de desarrollo es limitado.
- La solución debe ser razonablemente sencilla de desplegar.
- Conviene priorizar claridad y fiabilidad antes que complejidad técnica.
