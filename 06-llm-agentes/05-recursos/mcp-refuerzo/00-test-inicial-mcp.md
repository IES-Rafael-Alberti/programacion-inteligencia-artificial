# Test inicial: repaso de MCP

## Uso docente

Este material sirve como refuerzo opcional. Si el alumnado ya ha trabajado MCP con suficiente profundidad en el módulo donde corresponde, no hace falta impartirlo de nuevo: basta con recordarlo brevemente.

Si el test inicial muestra lagunas importantes, se puede usar este paquete como sesión corta de recuperación antes de continuar con agentes y herramientas.

## Preguntas rápidas

1. ¿Qué problema resuelve MCP cuando una aplicación de IA necesita usar herramientas externas?
2. ¿Qué diferencia hay entre el modelo, el cliente y las herramientas/servidores en una arquitectura MCP?
3. ¿Por qué conviene separar el contexto, las herramientas y la lógica de presentación?
4. ¿Qué riesgos aparecen cuando un agente puede llamar herramientas externas sin validación?
5. En el ejemplo `asistente_mcp.py`, identifica qué partes actúan como recuperación de contexto, herramienta externa e interfaz.

## Criterio de decisión

- Si la mayoría del grupo responde con solvencia, usar este recurso solo como recordatorio.
- Si aparecen dudas generalizadas, impartir el paquete como refuerzo guiado.
