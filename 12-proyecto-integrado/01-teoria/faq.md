# Preguntas frecuentes sobre las herramientas cloud del proyecto

## ¿Hace falta usar todas las herramientas del módulo?

No. Cada proyecto debe elegir solo las herramientas que realmente aporten valor.

## ¿Es obligatorio usar RAG?

No. Solo tiene sentido si el proyecto necesita consultar documentos o recuperar contexto externo.

## ¿Cuándo conviene usar una base vectorial?

Cuando el sistema necesita búsqueda semántica sobre documentos, fragmentos o elementos similares.

## ¿Cuándo puede quedarse corto el chunking clásico?

Cuando los documentos son largos, tienen mucha estructura interna o la respuesta depende de varias secciones relacionadas.

## ¿Cuándo compensa usar un modelo abierto?

Cuando importa el control sobre los datos, el coste por uso o el interés didáctico por aprender serving e inferencia.
