# 🐍 UD1 – Fundamentos de Programación para IA (Python I)

## 1. Problema de partida
En un instituto los alumnos participan en un concurso de matemáticas y los profesores necesitan **corregir rápidamente cientos de respuestas**.
Actualmente lo hacen a mano o con hojas de cálculo, pero:
- Se pierde mucho tiempo.
- Hay errores humanos.
- No se puede reutilizar el código para otros cálculos o actividades.

👉 Surge la necesidad de **automatizar estas tareas**.

---

## 2. Justificación de la unidad
Para resolver problemas de **automatización básica** necesitamos un lenguaje de programación: **Python**.
Es el estándar en IA y Data Science porque:
- Es sencillo de aprender y leer.
- Tiene gran cantidad de librerías científicas.
- Es usado en la industria y la investigación.

Esta unidad introduce los **fundamentos de Python**, base imprescindible para todo el módulo.

---

## 3. Objetivos específicos
- Conocer la sintaxis básica de Python.
- Usar variables y tipos de datos.
- Aplicar estructuras de control (condicionales y bucles).
- Definir y utilizar funciones simples.
- Entender la lógica detrás de un programa.

---

## 4. Contenidos
- Instalación y uso de Python.
- Jupyter Notebook / Google Colab.
- Tipos de datos y variables.
- Operadores.
- Condicionales (`if`, `elif`, `else`).
- Bucles (`for`, `while`).
- Funciones (`def`, parámetros, retorno).

---

## 5. Herramientas
- **Python 3.11+**
- **Jupyter Notebook** o **Google Colab**
- Editor: **VSCode** o **PyCharm Community**

---

## 6. Actividades propuestas
1. **Ejemplo guiado** → Programa que corrija una lista de operaciones matemáticas y muestre cuáles son correctas.
2. **Ejercicio individual** → Conversor de monedas (euros ↔ dólares).
3. **Ejercicio en parejas** → Generador de tablas de multiplicar usando bucles.
4. **Mini-proyecto grupal** → Simulador de notas de alumnos:
   - Entrada: lista de calificaciones.
   - Salida: nota media, nota máxima, número de aprobados y suspensos.

---

## 7. Evaluación (RA/CE vinculados)
- **RA1.a**: Caracterizar lenguajes de programación aplicados a IA.
- **Criterios**:
  - Escribe correctamente programas simples en Python.
  - Usa variables, estructuras de control y funciones adecuadamente.
  - Resuelve el problema planteado de forma lógica y reproducible.

📊 **Instrumentos de evaluación**
- Entrega de notebooks con ejercicios.
- Cuestionario rápido en Moodle.
- Observación del trabajo en clase.

## 9. Conclusión
Resolver problemas cotidianos (como corregir automáticamente una prueba) requiere aprender **los fundamentos de programación en Python**.
Esta base nos permitirá, más adelante, analizar datos, construir modelos de IA y desplegarlos.
